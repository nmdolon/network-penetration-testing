# Network Penetration Testing
Network penetration testing exercises conducted in isolated lab environments. Covers reconnaissance, scanning, enumeration, exploitation, and post-exploitation methodology.
## Exercises Completed

- Brute-force attack against WordPress login using Hydra — MrRobot Linux target

- SMB vulnerability assessment and exploitation methodology — Windows XP target (EternalBlue/MS17-010)
## Tools Used

Nmap · Hydra · Metasploit · arp-scan
